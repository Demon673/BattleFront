游戏人数 2到10人 ffa pvp
游戏为半回合制

区域的下半部分为战术放置区域，玩家可以在下半部分移动自己的单位
灰色的点为基地，为了迎双节可以用雪地天辉基地



基地拥有30点生命

游戏流程：
1 游戏开始时每位玩家拥有500金币
2 每个回合玩家会自动增加300金币
3 创建一个随机的基本单位花费100金币，可以卖出单位，单位售价40金币
4 每次回合开始，玩家会面对上一家玩家的部队，他们会按照该玩家在战术放置区域内摆放的位置镜像出现在你区域的上半部分
5 在厮杀结束后回合结束，没杀掉的敌人会冲向基地扣掉基地的一定生命

经济系统（存疑，矿工系统其实可以不要）：
1 游戏每回合的基础收入为300
2 玩家可以花费100金币购买一个黄金矿工，每个矿工每10s产出1枚金币 （需要一个类似水晶的二次货币吗？）
3 玩家可以升级矿工的采集效率
4 玩家可以使用屯野，屯野分为小野 中野 大野 远古野 肉山，均需要购买，购买后下一波进攻中会混入这些野怪，击杀野怪会获得金币

科技系统：
1 人口，人口限制了玩家可以拥有的兵和英雄数量
2 科技限制所有升级，每层科技会开放新的研究
   1 1层科技，开放更高人口上限，开放战旗，开放装备搜索（？），开放屯野
   2 2层科技，开放英雄搜索（直接出合成的1级英雄），开放更高的矿工采集效率，开放大野
   3 3层科技，开放远古野
   4 4曾科技，开放肉山，开放光环研究

战旗系统：
1 玩家可以花费1000金币购买战旗 战旗唯一 可以升级 受限制于科技
2 战旗种类：
	1 navi战旗： 		屠夫，陈，小精灵获得100%攻速加成
	2 ig战旗：  		濒死单位可以以灵魂形态战斗2s
	3 alliance战旗：	所有非英雄单位获得50%移动速度和攻击速度加成
	4 nb战旗：		每回合开始战斗时获得一个持续3s的减伤40%buff
	5 eg战旗：		玩家最贵的单位获得双倍伤害
	6 wings战旗：		玩家拥有的每个不同的兵可以给每个单位增加1%攻击力和1%攻速
	7 liquid战旗：	每死亡一个单位 剩下的单位获得4%吸血和4%攻速

装备系统：
1 开放式（所有装备随时可以购买）or随机式（花费金钱随机出装备）？
2 装备分为被动和主动 基本类似烽火


单位：
每个单位的魔法值为30点，每个技能需要消耗30点魔法。所有单位的魔法值恢复速度默认为：2点/s

基本单位

近战兵1-3级			技能：无（每级提高的属性比较好）
远程兵1-3级			技能：无（每级提高的属性比较好）
狼怪1-3级			技能：1级暴击效果、2级攻击光环、3级分裂效果
萨特1-3级			技能：1级净化、2级法力损毁、3级冲击波
枭兽1-3级			技能：1级魔抗光环、2级召唤飓风、3级专注光环
半人马1-3级			技能：1级战争践踏、2级攻速光环、3级移速光环
食人魔魔法师1-3级		技能：1级恢复光环、2级回蓝光环、3级冰霜护甲
熊怪1-3级				技能：1级重击、2级雷霆一击、3级嚎叫
树人1-3级				技能：1级缠绕、2级荆棘光环、3级淬毒武器


特殊单位
基本上为各种DOTA2英雄（少部分为厉害的野怪），由各种基本单位合成而来。玩家可以花费一定的金币来给特殊单位晋升，晋升会加强特殊单位各方面能力

斯温 = 
{
	配方 = 近战兵1级+狼怪3级
	技能 = 
	{
		战吼：主动，开战后主动使用；斯温的战吼激励着同伴作战，提升他们的护甲，持续一段时间。
		巨力挥舞：被动；斯温以巨力攻击，并对所有附近的敌方单位造成分裂攻击伤害。
	}
}
主宰 = 
{
	配方 = 近战兵2级+狼怪1级+食人魔魔法师1级
	技能 = 
	{
		治疗守卫：主动，开战后主动使用；召唤一个治疗守卫，按周围友军单位最大生命值的一定比例治疗他们。治疗守卫不可移动。
		剑舞：被动；主宰有一定概率在攻击中造成双倍伤害。
	}
}

灰烬之灵 = 
{
	配方 = 熊怪2级+枭兽1级+树人1级
	技能 = 
	{
		炎阳索：主动，自身150范围内有敌人时主动使用；灰烬之灵释放出火焰绳索来缠住附近的敌人，将他们困于原地并每秒造成一定伤害。
		烈火罩：被动；降低自己受到的魔法伤害，并且对灰烬之灵周围的敌人造成持续伤害。
	}
}
风暴之灵 = 
{
	配方 = 熊怪2级+远程兵1级+萨特1级
	技能 = 
	{
		残影：主动，攻击前主动使用；风暴之灵制造一个充满爆炸性能量的自身残影，持续一段时间，如果有人接近就会引爆造成伤害。
		超负荷：被动；施法时会产生电荷，在风暴之灵下次攻击时被释放，对附近敌人造成伤害并减速。
	}
}
大地之灵 = 
{
	配方 = 熊怪2级+枭兽1级+半人马1级
	技能 = 
	{
		石化：主动，攻击前主动使用；对附近的敌人造成伤害并且石化他们，被石化的敌人将会被降低攻速，持续一段时间。
		粉碎：被动；攻击有一定概率对周围的敌人造成伤害，如果目标被石化将会受到更多伤害。
	}
}

卓尔游侠 = 
{
	配方 = 远程兵2级+狼怪2级
	技能 = 
	{
		狂风：主动，攻击前主动使用；释放一阵狂风，沉默敌方单位并将其击退。击退距离与敌人和卓尔游侠的距离成反比。
		精准光环：被动；提高所有友方远程友军的攻击力百分比。
	}
}
风行者 = 
{
	配方 = 远程兵2级+半人马2级
	技能 = 
	{
		风行：主动，血量低于50%时主动使用；提升生命恢复速度并闪避一切物理攻击。
		集中火力：被动；当生命值为100%时，大幅增加自身攻击速度。
	}
}
